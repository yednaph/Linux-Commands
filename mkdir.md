<b> Some example usage of the `mkdir` command </b>

To create nested directories: `mkdir -p dir0/dir1/dir2`

To create dir0/dir1 and dir0/dir2 directories: `mkdir -p dir0/{dir1,dir2}`

To create the dir0/dir1, dir0/dir2, dir0/dir2/dir21 and dir0/dir2/dir22 directories: `mkdir -p dir0/{dir1,dir2/{dir21,dir22}}`
